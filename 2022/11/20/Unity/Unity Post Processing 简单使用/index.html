Unity Post Processing 简单使用

Unity Post Processing 简单使用

导入进项目

2020.1 Above URP Project

  • URP 项目自带 Post Processing，不需要额外添加

  • 在 Hierarchy 窗口创建 Global Volume

  • 在 Global Volume 中按下 New，创建配置文件

  • 记得要勾选主相机中的 Use Post Process

  • 通过 Add Override 添加效果

  • 如果是 URP，需要在 URP 资源中勾选 HDR 以开启更多效果

2019.4 Below

  • 在 Package Manager 中下载 Post Processing
  • 创建空物体并添加 Post Process Volume 组件
  • 创建配置文件，拖拽到 Volume 上
  • 勾选 isGlobal，全局响应
  • 在创建的空物体的 Layer 新建一个专门用于后处理的 Layer 并选择
  • 选中主相机，添加 Post Process Layer 组件，并且选择后处理的 Layer

相机和电影视觉效果

  • Bloom：炫光
  • Chromatic Aberration：色差
  • Film Grain：胶片颗粒
  • Vignette：暗角
  • Depth of Field：景深
  • Motion Blur：运动模糊
  • Lens Distortion：镜头畸变
  • Panini Projection：帕尼尼效果
  • (2019)Ambient Occlusion：环境光遮蔽

颜色效果

  • Tone Mapping：色调映射
  • White Balance：白平衡
  • Color Adjustment：色彩调整
  • Split Tone：色调分离

​

  • Channel Mixer：通道混合器
  • Color Curves：颜色曲线
  • Shadows，Midtones，Highlights：阴影，中间色调，高光
  • Life，Gamma，Gain：提升，伽马，增益
  • Color Lookup Table：颜色查找

一些有关设定

  • Dithering：抖色
  • Anti-Aliasing：抗锯齿

脚本控制 Post Processing

  • 命名空间：UnityEngine.Rendering.PostProcessing

物体上挂载 Volume 时操作对象

using UnityEngine.Rendering.PostProcessing

private PostProcessVolue volume;
private Vignette vignette;

private void Start()
{
    volue=GetComponent<PostProcessVolume>();
    volume.profile.TryGetSetting(out vignette();
}
                                 
private void AddVignette()
{
    if(Input.GetKeyDown(KeyCode.Space))
    {
        vignette.intensity.value = 1.0f;
        //vignette.intensity.Override(1.0f);
    }
}

没有 Volume 时操作对象

using UnityEngine.Rendering.PostProcessing

private PostProcessVolue volume;
private Vignette vignette;

private void Start()
{
    vignette = ScriptableObject.CreateInstance<Vignette>();
    vignette.enabled.Overide(true);
    vignette.intensity.Overide(1.0f);
    
    volume = PostProcessManager.instance.QuickVolume(gameObject.layer,1,vignette);
}
                                 
private void AddVignette()
{
    ///
}

private void OnDestroy()
{
    RuntimeUtilities.DestroyVolume(volume,true,true);
}
